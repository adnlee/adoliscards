
# CardVault V2.1

## Install

1. Back up your current `adoliscards` folder.
2. Extract this ZIP.
3. Copy `app.py`, `requirements.txt`, `.gitignore`, and `.streamlit/config.toml`
   into your existing GitHub repository folder.
4. Replace the old files when Windows asks.
5. Keep your existing `.streamlit/secrets.toml`.
6. No new Supabase migration is required if V2 is already installed.
7. Run:

   ```
   py -m pip install -r requirements.txt
   py -m streamlit run app.py
   ```
